# Lume Landing Page

Projeto desenvolvido com foco em prática de CSS Grid, Flexbox e responsividade (mobile-first).

## 💡 Objetivo
Treinar construção de layout moderno utilizando boas práticas de CSS.

## 🚧 Observações
O design não foi o foco principal. Este projeto foi utilizado como exercício prático para consolidar conhecimentos.

## 🛠️ Tecnologias
- HTML
- CSS (Grid, Flexbox, variáveis CSS)
